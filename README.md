# Selenium UI Automation Practice 

This project is created for practicing major UI automation scenarios using Selenium WebDriver.

## Application Under Test

Practice Website:  
http://uiplayground.techwithjatin.com/

## Objective

The goal of this framework is to automate and practice common Selenium UI operations such as:

- Login functionality
- Button interactions
- Text box handling
- Dropdown selection
- Checkbox and radio button handling
- Alerts and popups
- Frames and windows
- Tables and web grids
- Mouse hover actions
- Drag and drop
- Waits and synchronization
- File upload/download
- JavaScript execution
- Navigation commands
- Form validation
- Dynamic elements handling

## Tech Stack

- Java
- Selenium WebDriver
- TestNG / JUnit
- Maven
- Git
- GitHub
- CI/CD Pipeline
